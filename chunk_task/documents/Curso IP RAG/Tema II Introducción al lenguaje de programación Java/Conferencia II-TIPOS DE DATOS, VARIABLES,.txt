**Diapositiva 12: Variables (Visibilidad - Regla)**
*   **Página 12:** Indica que, generalmente, las variables dentro de llaves `{}` solo son **visibles** dentro de esas llaves.

**Diapositiva 13: Tipo de Datos (Introducción)**
*   **Página 13:** Explica que todos los datos tienen un **tipo de dato** asociado, el cual define los valores y las operaciones posibles con la variable.

**Diapositiva 14: Tipo de Datos (Jerarquía)**

**Diapositiva 15: Tipo de Datos - Numéricos (Introducción)**
*   **Página 15:** Define los tipos de datos **numéricos** como los que representan valores escalares y permiten operaciones aritméticas.

**Diapositiva 16: Tipo de Datos - Numéricos (Enteros)**
*   **Página 16:** Describe los **enteros** como números sin decimales y que las operaciones no resultan en decimales. (Ej: 25, -96, 0).

**Diapositiva 17: Tipo de Datos - Numéricos (Reales)**
*   **Página 17:** Describe los **reales** como números con punto decimal y que las operaciones pueden resultar en decimales. (Ej: 12.3, -78.56, 45.0).

**Diapositiva 18: Tipo de Datos - Lógicos**
*   **Página 18:** Explica los tipos de datos **lógicos** (booleanos), que solo pueden ser "verdadero" o "falso".

**Diapositiva 19: Tipo de Datos - Carácter**
*   **Página 19:** Indica que los tipos de datos **carácter** son equivalentes al código ASCII para representar letras y símbolos.

**Diapositiva 20: Tipo de Datos - Cadena de Caracteres (Introducción)**
*   **Página 20:** Describe las **cadenas de caracteres** como texto, incluyendo nombres y direcciones.

**Diapositiva 21: Tipo de Datos - Cadena de Caracteres (Limitaciones)**
*   **Página 21:** Aclara que los números en cadenas pierden sus propiedades matemáticas y que estas se escriben entre comillas.

**Diapositiva 22: Tipos de Datos - Java (Parte 1)**
*   **Página 22:** Muestra una **tabla con tipos primitivos de Java** y sus rangos: `boolean`, `byte`, `short`, `int`, y `long`.

**Diapositiva 23: Tipos de Datos - Java (Parte 2)**

**Diapositiva 24: Constantes (Definición)**
*   **Página 24:** Define las **constantes** como valores que no cambian durante la ejecución del programa.

**Diapositiva 25: Constantes (Almacenamiento)**
*   **Página 25:**  Explica que las constantes ocupan un espacio **fijo en la memoria**.

**Diapositiva 26: Constantes (Java)**
*   **Página 26:**  Muestra que en Java, la palabra reservada `final` define una **constante**. (Ej: `final double PI`).

**Diapositiva 27: Identificadores (Definición)**
*   **Página 27:**  Define los **identificadores** como los nombres que se usan para referirse a las variables, constantes, funciones, etc.

**Diapositiva 28: Identificadores (Reglas - Longitud)**
*   **Página 28:** Indica que los identificadores no pueden tener más de **31 caracteres**.

**Diapositiva 29: Identificadores (Reglas - Sensibilidad a Mayúsculas)**
*   **Página 29:** Indica que los identificadores **distinguen** entre mayúsculas y minúsculas (case-sensitive).

**Diapositiva 30: Identificadores (Reglas - Primer Carácter)**
*   **Página 30:** Aclara que el primer carácter debe ser una **letra o `_`** (guion bajo), no un número.

**Diapositiva 31: Identificadores (Reglas - Caracteres Válidos)**
*   **Página 31:**  Enumera los **caracteres válidos** en un identificador: letras minúsculas (a-z), mayúsculas (A-Z), dígitos (0-9) y guion bajo (`_`).

**Diapositiva 32: Identificadores (Reglas - Caracteres Inválidos)**
*   **Página 32:** Indica que **no se permiten** espacios en blanco ni caracteres especiales como `#`, `$`, `%`, `*`, `&`.

**Diapositiva 33: Identificadores (Palabras Reservadas)**
*   **Página 33:**  Advierte que hay **palabras reservadas** en el lenguaje que no pueden usarse como identificadores.

**Diapositiva 34: Identificadores (Ejemplos)**

**Diapositiva 35: Identificadores (Buenas Prácticas)**
